# Frontend — Responsive Webpage

## Project Overview
This project is a fully responsive webpage built based on the provided Figma design (**Website Design [FIA]**).

The goal of this assignment is to demonstrate:
- Strong frontend fundamentals
- Clean UI implementation
- Smooth animations
- Responsive design
- Accessibility best practices

---

## Features
  **UI & Design:**
  - Pixel-perfect implementation based on Figma
  - Clean and modern layout
  - Fully responsive across all screen sizes

  **Animations:**
  - Animated background behind astronaut image
  - Smooth transition effects
  - Timeline slider animation

  **Interactions:**
  - Cursor hover effects on images
  - Interactive UI components
  - Smooth scrolling experience

  **Responsiveness:**
  - Mobile-first approach
  - Optimized for Retina & standard displays
  - Adaptive layout for tablets and desktops

  **Accessibility:**
  - Semantic HTML elements
  - Keyboard navigation support
  - Proper alt text and ARIA roles

## ⚙️ Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/Prity25-coder/WebPage_H2S.git
cd WebPage_H2S
``` 

### 2. Install dependencies 
    npm install
    ```    

### 3. Run the project 
    npm run dev
    ```    

### 3. Access the web page 
    Open your browser and go to [http://localhost:3000]
